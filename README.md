<h3 align="center">ارزیابی تعالی سازمانی بر پایه مدل EFQM 2025</h3>

---

### 📘 درباره این مخزن  
این مخزن به عنوان **پایگاه ارزیابی مدل تعالی EFQM 2025** ایجاد شده است و شامل:
- مستندات رسمی ارزیابی  
- شواهد مرتبط با معیارها و زیرمعیارهای EFQM  
- پروژه‌های بهبود استخراج‌شده بر پایه منطق RADAR  
- ابزارها و قالب‌های ارزیابی برای ارزیابان و مشاوران تعالی  

---

### 🧭 ساختار پوشه‌ها
📂 .github/workflows → فایل‌های اتوماسیون و CI
📂 assets/ → تصاویر، بنرها، لوگو و نمودارها
📂 assessments/ → مستندات و گزارش‌های ارزیابی
📂 improvement-projects/ → پروژه‌های بهبود و اولویت‌ها
📂 reference/ → منابع و استانداردهای EFQM
README.md → صفحه اصلی مخزن
